msr
===

一个豆瓣登录机器人，用于自动化下载陌声人广播

===

回家的火车上应该会特别无聊，所以想下一些豆瓣陌声人广播在火车上听，但是链接太多，而且歌曲Id都是没有规律的，所以就写了这一个爬虫，模拟登陆豆瓣后，获取到需要下载的歌曲的所有url，然后使用多线程实现自动化下载。一个晚上就下完了500个广播大约10G的内容。

===

环境：python3 + windows
linux环境下需要按着代码注释修改一下即可

===

不足：
1. 没有破解验证码，因为只需登录一次就可以下载所有内容。在每次运行时会弹出图片，人工识别填写。

优点：
1. 如果程序出错退出之后，重新启动可以继续下载。
